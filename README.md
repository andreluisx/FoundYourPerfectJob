## :clipboard: Como Executar o Projeto

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
3. Execute o projeto:
  ```bash
  npm start
```
4. Abra no navegador pelo endereço http://localhost:3000.
